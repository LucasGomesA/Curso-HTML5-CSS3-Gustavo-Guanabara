Desafio dos Emoji 😜

Para colocar emoji usa-se (&#x) antes do código do emoji (Os códigos podem ser pegos na internet em sites como emojipédia).

Carinhas

  • 😎 Código U+1F60E
  • 🤠 Código U+1F920
  • 🥵 Código U+1F975

Mãos

  • 👉 Código U+1F449
  • 🖕 Código U+1F595
  • 👊 Código U+1F44A

Animais

  • 🙈 Código U+1F648
  • 🙉 Código U+1F649
  • 🙊 Código U+1F64A

Esportes

  • ⚽ Código U+26BD
  • 🏀 Código U+1F3C0
  • 🏐 Código U1F3D0

Comidas

  • 🍕 Código U+1F355
  • 🍔 Código U+1F354
  • 🍟 Código U+1F35F

Objetos

  • 💶 Código U+1F4B6
  • 💵 Código U+1F4B5
  • 💰 Código U+1F4B0
